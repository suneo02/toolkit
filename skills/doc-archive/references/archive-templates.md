# 归档模板速查

## 新建文件头

```markdown
# [对话主题]

**创建时间**：YYYY-MM-DD HH:MM
**主要参与者**：[用户角色/姓名]
**核心目标**：[1-2句话说明对话目的]
**预计产出**：[期望得到的成果，如文档、代码、方案等]

---
```

## 记录条目

```markdown
## [序号] [时间戳YYYY-MM-DD HH:MM] [会话标签]【当前使用的AI模型】

**💬 用户输入**：
[完整记录用户的提问或指令，保留原始语气和关键细节]

**🤖 本次响应处理**：

- **分析要点**：[解读用户需求的核心要点]
- **执行动作**：[具体采取了哪些行动]
- **产出内容**：[生成了什么具体成果，尽量量化]
- **注意事项**：[需要特别关注的点或限制条件]

**📊 关键结论/成果**：
[本次对话中最重要的1-3个结论或产出物]

**🔗 相关上下文**：
[与历史记录的关联点，或对后续讨论的提示]
```

## 结尾提示

```
---

📝 **对话记录已更新**

**文件状态**：[新建/追加] → `文件名.md`
**本次记录摘要**：[1句话总结记录内容]
**下次延续提示**：如需继续此话题，请提及"[关键词]"
```

## 命名规则

- 仅允许使用字母、数字、下划线 `_` 与连字符 `-`。
- 示例：`topic_20251212.md`
