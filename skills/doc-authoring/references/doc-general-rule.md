# General Documentation Rules

## Quick Standards

- **Language**: All content in Chinese.
- **Visuals First**: Diagrams/Tables > Text.
- **Length**: Max 1 page (~120 lines). Focus on "What", "Why", "How".
- **Location**: `docs/` folder. Use relative paths.
- **Links**: Bi-directional `@see` links between code and docs.

## Doc Types

| Type            | Purpose            | Key Content                    |
| --------------- | ------------------ | ------------------------------ |
| **README**      | Quick overview     | 作用一句话、结构树、关联文档   |
| **Requirement** | "What to do"       | 目标、页面流程、数据、限制     |
| **Design**      | "How to present"   | 信息架构、页面蓝图、状态与流程 |
| **Spec**        | "How to implement" | 任务拆解、JSON 实施清单        |
