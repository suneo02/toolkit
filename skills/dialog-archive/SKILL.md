# 对话归档

## 默认流程（少问问题）

- 归档范围：本次对话
- 模式：默认新建；仅当用户明确要求“继续/追加/延续”或给出已有文件路径时才追加
- 参与者：用户
- 模型：直接使用当前模型标识，不询问
- 输出目录：`docs/ai-chat/archive/`
- 文件名：`dialog_<YYYYMMDD_HHMM>.md`；若能从指令提取主题关键词，使用 `<topic>_dialog_<YYYYMMDD_HHMM>.md`
- 语言：记录文本必须中文

## 执行步骤

1. 使用 `date "+%Y-%m-%d %H:%M"` 获取时间。
2. 自动生成标题与文件名；无主题时标题为“对话归档”。
3. 写入：文件头、单条对话记录、确认块。
4. 返回生成文件路径。

## 模板与追加规则

- 模板、命名规则、对话类型判定与追加规范见 `references/archive-templates.md`，仅在需要输出模板或调整格式时读取。

## 仅在用户明确要求时才扩展

- 追加到已有文件：先读取完整文件，再追加并写回。
- 归档来源不是本次对话：按需读取 spec 文档或运行 `scripts/git_history_query.py`。
- 指定输出目录或文件名：使用用户要求，不再询问其他参数。
